Each expense category maps to a specific box on your HMRC Self Assessment **SA103F** form. Choosing the correct category ensures your expenses are reported correctly.

| Category | SA103F box |
|----------|------------|
| Cost of goods | Box 17 |
| Staff costs | Box 19 |
| Travel | Box 20 |
| Premises | Box 21 |
| Office costs | Box 23 |
| Professional fees | Box 28 |
| Other expenses | Box 30 |
