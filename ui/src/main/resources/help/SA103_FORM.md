The SA103 is the supplementary page for self-employment income in your Self Assessment tax return.

There are two versions:

- **SA103F (Full)** — for businesses with turnover over **£90,000** (the VAT-registration threshold since April 2024).
- **SA103S (Short)** — for simpler businesses.

This software uses the full-form format for complete, accurate reporting.
