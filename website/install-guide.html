Installation Guide

Get up and running in minutes. No Java or developer tools required.

Installing on Windows

System Requirements

  • Windows 10 or later (64-bit)
  • 200 MB free disk space
  • No Java installation required (bundled with the installer)

Download & Install

  1. Download SelfEmploy-x.x.x.msi from the latest release
  2. Double-click the downloaded .msi file
  3. If Windows SmartScreen shows a warning, click "More info" then "Run anyway"
    (The installer is not yet code-signed. This is safe for open-source software you downloaded from GitHub.)
  4. Follow the installation wizard
  5. Launch SelfEmploy from the Start Menu or desktop shortcut

One-Line Install (PowerShell)

.\install.ps1 -Install

This downloads and installs the latest release automatically.

Uninstall

Open Settings > Apps > Installed apps, find "SelfEmploy", and click Uninstall.

Troubleshooting

SmartScreen blocks the installer

Click "More info" in the SmartScreen dialog, then click "Run anyway". This happens because the installer isn't code-signed yet. The software is open source and can be audited on GitHub.

Application won't start

Try running SelfEmploy from the Start Menu. If it doesn't appear, check that the installation completed successfully in Settings > Apps. You can also try reinstalling.

Installing on macOS

System Requirements

  • macOS 12 (Monterey) or later
  • Apple Silicon (M1/M2/M3) or Intel Mac
  • 200 MB free disk space
  • No Java installation required (bundled with the installer)

Download & Install

  1. Download SelfEmploy-x.x.x.dmg from the latest release
  2. Double-click the downloaded .dmg file
  3. Drag SelfEmploy to the Applications folder
  4. Open SelfEmploy from Applications
  5. If macOS Gatekeeper shows a warning:
    Right-click (or Control-click) the app, select "Open", then click "Open" in the dialog
    (You only need to do this the first time. The app is not yet notarized by Apple.)

One-Line Install (Terminal)

./install.sh --install

This downloads and installs the latest release automatically.

Uninstall

Drag SelfEmploy from the Applications folder to the Trash.

Troubleshooting

Gatekeeper blocks the app

Right-click the app and select "Open" instead of double-clicking. In the dialog that appears, click "Open". You only need to do this once. This happens because the app is not yet notarized with Apple.

"SelfEmploy is damaged" error

This can occur if the download was incomplete. Delete the DMG, re-download it, and try again. If the issue persists, try: xattr -cr /Applications/SelfEmploy.app

Installing on Linux

System Requirements

  • Ubuntu 22.04+ / Debian 12+ / Fedora 38+ / RHEL 9+
  • x86_64 architecture
  • 200 MB free disk space
  • No Java installation required (bundled with the installer)

Download & Install (Debian/Ubuntu)

  1. Download SelfEmploy-x.x.x.deb from the latest release
  2. Install with: 
    sudo dpkg -i SelfEmploy-*.deb
  3. Launch from the application menu or run SelfEmploy from the terminal

Download & Install (Fedora/RHEL)

  1. Download SelfEmploy-x.x.x.rpm from the latest release
  2. Install with: 
    sudo rpm -i SelfEmploy-*.rpm
  3. Launch from the application menu or run SelfEmploy from the terminal

One-Line Install (Terminal)

./install.sh --install

This detects your package manager and downloads the appropriate installer automatically.

Uninstall

# Debian/Ubuntu
sudo dpkg -r selfemploy

# Fedora/RHEL
sudo rpm -e selfemploy

Troubleshooting

Missing dependencies

If dpkg reports missing dependencies, run: sudo apt-get install -f

Application doesn't appear in menu

Log out and log back in, or run SelfEmploy directly from the terminal.
